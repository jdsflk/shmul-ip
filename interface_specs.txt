Felhasználás:
	MMIO perifériavezérlő:
		operand_size: (szintézis paraméter) generic integer 5 bit (hány bitesek az operandusok)  
		operand:      generic integer 16 bit (szorzótényező érkezését jelzi)
		intr_en:      generic integer 1 bit  (intr enable állítását jelzi)
		product_1:    generic integer 16 bit (végeredmény 0-15  innen olvasható ki)
		product_2:    generic integer 16 bit (végeredmény 16-31 innen olvasható ki)
		product_3:    generic integer 16 bit (végeredmény 32-47 innen olvasható ki)
		product_4:    generic integer 16 bit (végeredmény 48-63 innen olvasható ki)
		ready:      generic integer 1 bit  (készen van-e a szorzás, ha nincs engedélyezve a megszakítás)
	user logic:
		start: in boolean
		intr_enable: in boolean
		operand_size: in integer 5 bit 
		op_1: in integer 32 bit
		op_2: in integer 32 bit
		result: out integer 64 bit
		ready: out boolean
		start_ack: out boolean
